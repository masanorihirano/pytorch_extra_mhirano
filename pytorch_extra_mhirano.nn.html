class pytorch_extra_mhirano.nn.attention.DotProductAttention(qdim: int, output_dim: Optional[int] = None, dropout: float = 0.0, transform: bool = True, bias: bool = True, same_embd: bool = True, add_bias_kv: Optional[bool] = None, kdim: Optional[int] = None, vdim: Optional[int] = None, batch_first: bool = True, scaled: bool = False)[source]¶

Bases: torch.nn.modules.module.Module

DotProductAttention.

\[ \begin{align}\begin{aligned}\mathrm{DotProductAttention}(Q, K, V) &=& \mathrm{softmax}(qk^T) v\\q &=& QW_1 + b_1\\k &=& KW_2 + b_2\\v &=& VW_3 + b_3\end{aligned}\end{align} \]
Parameters

  • qdim – dimension of the model, i.e., dimension of Q

  • hidden_dim – dimension of hidden layer, i.e., dimension of q, k, v. Default: 512

  • output_dim – dimension of output layer, i.e., dimension of output. Default: None

  • dropout – a Dropout layer on attn_output_weights. Default: 0.0.

  • transform – q = Q, k = K, v = V if it is False. Default: True

  • bias – add bias as module parameter. Default: True.

  • same_embd – W1 = W2 = W3, b1 = b2 = b3 if it is True. Default: True

  • add_bias_kv – add bias to the key and value sequences at dim=0.

  • kdim – total number of features in key. Default: None.

  • vdim – total number of features in key. Default: None.

  • Note – if kdim and vdim are None, they will be set to embed_dim such that

  • query –

  • key –

  • features. (and value have the same number of) –

Examples::
>>> attn = DotProductAttention(query_dim)
>>> attn_output, attn_output_weights = attn(query, key, value)
